## Objective

Прочитать вместе с Rename-AIBus-Process-To-TaskBus.md перед
выполнением (та задача ещё не в active/, эта — маленькое дополнение
к ней, не отдельный шаг).

Полное имя в документации — "Task Bus" (заголовки, названия файлов:
TASK_BUS_V4.md и т.п.) — без изменений от исходного Step Card.

Добавь: краткая разговорная форма — "TBus" — допустима в тексте
внутри документов как сокращение после первого полного упоминания в
файле (например: "Task Bus (TBus) — ..." при первом упоминании в
файле, дальше можно использовать TBus). Не обязательно менять уже
написанные тобой в рамках этой задачи файлы ради этого — просто
взять на заметку для будущих документов и ARP/REVIEW переписки.

## Scope / Validation / Output / Stop Condition

Как в основном Step Card Rename-AIBus-Process-To-TaskBus.md — это
не отдельная задача, просто уточнение терминологии.
