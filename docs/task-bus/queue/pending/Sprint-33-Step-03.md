## Контекст

Сейчас Sidebar имеет несколько разделов: "Книга", "Серии", "Персонажи", "Идеи", "Корзина".
Каждый может быть развёрнут или свёрнут независимо. Это приводит к скролу в боковой панели, 
когда все разделы развёрнуты одновременно.

Улучшение: "Smart Collapse" — когда пользователь кликает на заголовок раздела 
("Книга", "Серии", и т.д.), этот раздел разворачивается, а все остальные сворачиваются.
Это похоже на аккордеон (accordion), но для секций, не для элементов внутри.

## Scope

### Allowed paths (ТОЛЬКО):

- apps/studio/src/components/Sidebar.tsx (переработать логику collapse/expand разделов)
- apps/studio/src/app/page.tsx (добавить state для активного раздела, передать в Sidebar)

### Forbidden paths:

## Objective

Реализовать "аккордеон"-режим для разделов Sidebar:

1. **State management (page.tsx):**
   - Добавить `expandedSectionId?: string | null` (какой раздел развёрнут сейчас)
   - Функция `onToggleSectionExpanded(sectionId: string)` → раскрывает этот раздел, закрывает остальные

2. **Логика:**
   - Разделы: "books", "series", "characters", "ideas", "trash" (ID каждого)
   - При загрузке: по умолчанию "books" развёрнута (или состояние из localStorage)
   - При клике на заголовок раздела:
     - Если раздел уже развёрнут → свернуть его (закрыть все)
     - Если раздел свёрнут → развернуть его (закрыть остальные)

3. **Sidebar UI:**
   - Заголовок раздела (h2 "Книга", "Серии" и т.д.) становится clickable
   - Добавить иконку collapse/expand справа от заголовка (▾/▸)
   - Клик на заголовок вызывает onToggleSectionExpanded

4. **Состояние:**
   - Persist в localStorage: "literary-studio:expanded-section" → sectionId
   - При загрузке: восстановить из localStorage или "books" по умолчанию

## Rules

1. Нет множественного expand: одновременно развёрнут только один раздел
2. Клик на заголовок toggle (развёрнут → свёрнут, свёрнут → развёрнут)
3. Содержимое развёрнутого раздела полностью видимо (не вложенный collapse внутри)
4. Collapse/expand плавный переход (transition класс для высоты)
5. На мобильных: может быть выключено (? или всегда accordion)

## Validation

Живая проверка в браузере:

1. **При загрузке:**
   - Раздел "Книга" развёрнут по умолчанию
   - Остальные свёрнуты (видны только заголовки)

2. **Клик на "Серии":**
   - "Серии" разворачивается
   - "Книга" сворачивается
   - Список серий видим

3. **Клик на "Персонажи":**
   - "Персонажи" разворачивается
   - "Серии" сворачивается
   - И т.д.

4. **Toggle свёрнутого:**
   - Клик на свёрнутый раздел → развёртывается (другие закрываются)

5. **Persist:**
   - После refresh — развёрнут тот же раздел, что был раньше
   - localStorage содержит expanded-section

6. **Scroll:**
   - Боковая панель больше не скролится внутри (весь контент одного раздела видим одновременно)
