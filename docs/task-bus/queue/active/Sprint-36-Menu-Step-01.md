## Требование

Реализовать функциональное меню "Файл" с поддержкой основных операций: создание новой книги, создание серии, экспорт, выход. В Header.tsx уже есть placeholder структура меню и localization строки, но кнопки либо disabled, либо не подключены к обработчикам.

**Текущее состояние:**
- ❌ "Новая книга" — функция работает частично (onCreateBook вызывается)
- ❌ "Новая серия" — обработчик не подключен
- ❌ "Экспортировать" — обработчик не подключен
- ❌ "Выход" — обработчик не подключен (logout)

**Должно быть:**
- ✅ Все пункты меню "Файл" функциональны
- ✅ Клики на пункты вызывают корректные действия
- ✅ Меню закрывается после выбора пункта
- ✅ Клавиатурные ярлыки готовы (Ctrl+N, Ctrl+Shift+S, Ctrl+E)

## Что нужно сделать

### Part A: Подключение обработчиков в Header.tsx

1. **Для "Новая книга":**
   - Убедиться, что `onCreateBook?.()` вызывается при клике
   - Пункт меню не должен быть disabled
   - Добавить aria-label с указанием Ctrl+N

2. **Для "Новая серия":**
   - Добавить параметр `onCreateSeries?: () => void` в `HeaderProps`
   - Вызывать `onCreateSeries?.()` при клике на пункт меню
   - Добавить aria-label: "Новая серия (Ctrl+Shift+N)"

3. **Для "Экспортировать":**
   - Обработчик `onExportBook` уже передается, но нужно убедиться что вызывается
   - Если есть несколько книг, показать submenu для выбора какую экспортировать
   - Или открыть диалог "Экспортировать активную книгу"

4. **Для "Выход":**
   - Вызывать `onLogout?.()` при клике
   - Перед выходом показать подтверждение (опционально)

### Part B: Обновление page.tsx для новых обработчиков

1. **Добавить функцию для создания серии:**
   ```typescript
   const createSeries = (title: string) => {
     const newSeries = { id: generateId(), title, books: [] };
     workspace.series.push(newSeries);
     saveWorkspace();
   };
   ```

2. **Добавить функцию для экспорта:**
   - Если книг несколько, открыть диалог выбора
   - Если активна одна книга, экспортировать её
   - Вызвать существующий обработчик `handleExportBook`

3. **Передать новые обработчики в Header:**
   ```typescript
   onCreateSeries={() => {
     // Открыть диалог ввода названия серии
     setNewSeriesDialogOpen(true);
   }}
   onExportBook={handleExportBook}
   ```

### Part C: Диалоги и модалы

1. **Убедиться что существует диалог для создания серии:**
   - Проверить `NewSeriesDialog.tsx`
   - Если не полностью реализован, добавить поле ввода названия

2. **Экспорт:** Использовать существующий `ExportDialog.tsx`

### Part D: Localization

- Все строки уже в `public/locales/ru/common.json` и `public/locales/en/common.json`
- Проверить что нет hardcoded текста в меню File
