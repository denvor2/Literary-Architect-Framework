## Технические детали

### CSS переменные

Переменные добавлены на уровне `:root` и переопределены в `@media (prefers-color-scheme: dark)`:

```css
:root {
  /* Light mode */
  --background: #ffffff;
  --foreground: #171717;
  --border: #e4e4e7;
  --input-bg: #ffffff;
  --input-border: #d4d4d8;
  --muted: #f4f4f5;
  --muted-foreground: #71717a;
  --accent: #2563eb;
  --accent-foreground: #ffffff;
  --success: #16a34a;
  --warning: #ea580c;
  --error: #dc2626;
  --info: #0284c7;
}

@media (prefers-color-scheme: dark) {
  :root {
    /* Dark mode */
    --background: #09090b;
    --foreground: #f4f4f5;
    --border: #27272a;
    --input-bg: #18181b;
    --input-border: #3f3f46;
    --muted: #27272a;
    --muted-foreground: #a1a1aa;
    --accent: #3b82f6;
    --accent-foreground: #09090b;
    --success: #22c55e;
    --warning: #f97316;
    --error: #ef4444;
    --info: #06b6d4;
  }
}
```

### Выбор цветов

**Dark Mode Background (#09090b vs #0a0a0a):**
- Использован #09090b вместо #0a0a0a для большей совместимости с OLED
- Оба цвета практически идентичны визуально, но #09090b стандартизирован в zinc scale

**Dark Mode Foreground (#f4f4f5 vs #ededed):**
- Улучшено с #ededed на #f4f4f5 для лучшего контраста
- #f4f4f5 — это zinc-100, соответствует Tailwind's default scale

**Blue Accent (#3b82f6 vs #2563eb):**
- #3b82f6 (blue-500) более яркий и видимый в темном режиме
- #2563eb (blue-600) подходит для light режима
- Оба остаются узнаваемыми и консистентными в brand identity
