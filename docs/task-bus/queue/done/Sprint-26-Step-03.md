type: ui

## Контекст

Сейчас Ideas (заметки) показываются в IdeasPanel полностью развёрнутыми. Когда заметок много,
это занимает много места. Нужно показывать по одной строке, а полный текст — при клике или
в отдельной карточке.

## Scope

Allowed paths:
- apps/studio/src/components/IdeasPanel.tsx
- apps/studio/src/app/globals.css (если нужны новые стили)

Forbidden paths:
- Domain model (Idea тип не меняется)
- API endpoints
- Других компонентов

## Rules

- Заметки отображаются списком, по одной строке каждая (truncated)
- При клике на заметку: либо развернуть строку (expand), либо показать modal/tooltip с полным текстом
- Сохранить возможность редактирования и удаления (иконки остаются видны)
- Дата создания остаётся видна
- Без новых зависимостей — только Tailwind классы (truncate, line-clamp и т.д.)

## Validation

- npx tsc --noEmit - чисто
- npx eslint src - чисто
- npm run build - чисто
- Живая проверка (npm run dev):
  1. Создать несколько заметок с длинными текстами
  2. Проверить, что каждая заметка показывается одной строкой (truncated)
  3. Кликнуть на заметку — показать полный текст
  4. Проверить edit/delete кнопки видны и работают
