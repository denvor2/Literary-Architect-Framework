## Что Сделано

Реализована статистика слов и знаков в подвал страницы. Пользователь видит в реальном времени количество слов, знаков с пробелами и знаков без пробелов для активной книги. Функция работает на всех устройствах (мобильная, планшет, десктоп) и поддерживает темный режим.

### Компоненты

1. **StatsFooter.tsx** (новый компонент)
   - Отображает три метрики: "Слов", "Знаков", "Без пробелов"
   - Форматирует числа с разделителем тысяч (1,234)
   - Поддерживает темный режим через `dark:` классы Tailwind
   - Отзывчивый дизайн для всех размеров экрана

2. **page.tsx** (обновлено)
   - Добавлена функция `calculateStats()` для вычисления статистики
   - Функция суммирует текст всех сцен активной книги
   - Логика подсчета:
     - **Слова**: `text.trim().split(/\s+/).filter(Boolean).length`
     - **Знаки с пробелами**: `text.length`
     - **Знаки без пробелов**: `text.replace(/\s/g, '').length`
   - Компонент `StatsFooter` добавлен в мобильный и десктоп-макеты
   - На мобильном: после `MobileBottomNav`
   - На десктопе/планшете: после основного контейнера, перед `DeveloperTools`

3. **word-stats.spec.ts** (E2E тесты)
   - Тест: footer отображается при загрузке
   - Тест: пустая книга показывает 0 по всем метрикам
   - Тест: статистика обновляется при добавлении текста
   - Тест: статистика обновляется при редактировании текста
   - Тест: статистика корректно суммирует несколько сцен
   - Тест: форматирование чисел с разделителем тысяч
   - Тест: адаптивность на мобильных устройствах
   - Тест: поддержка темного режима
   - Тест: персистентность при переключении между книгами

4. **CRITICAL_FEATURES.md** (обновлено)
   - Добавлены 5 функций в раздел Sprint-38
   - Обновлена статистика coverage (21 функция вместо 16)
