name: "Requisites UI: выделить фоном, иконка сворачивания, кнопка Развернуть в свернутом состоянии"
type: ui

## Контекст

Блок реквизитов (requisites — название книги, жанр, аннотация) нужно визуально выделить:
добавить фоновый цвет, сделать сворачивание более очевидным (иконка вместо текста), 
и в свернутом состоянии показать кнопку "Развернуть".

## Scope

Allowed paths:
- apps/studio/src/components/EditorArea.tsx (блок реквизитов)
- apps/studio/src/app/globals.css (если нужны новые стили)

Forbidden paths:
- Domain model
- API endpoints
- Других компонентов

## Rules

- Блок реквизитов получает фоновый цвет (zinc-50/dark:zinc-950 в стиле UI системы)
- Кнопка "Свернуть" заменяется на иконку (например, ChevronDown/ChevronUp из lucide-react)
- В свернутом состоянии блок остаётся видимым (не исчезает), но показывает только заголовок
  "Реквизиты книги" + кнопка "Развернуть" (обратный текст к "Свернуть")
- Сворачивание остаётся эфемерным UI-состоянием (не сохраняется в БД)
- Тёмный режим: фоновый цвет адаптируется (zinc-50 light / zinc-950 dark)

## Validation

- npx tsc --noEmit - чисто
- npx eslint src - чисто
- npm run build - чисто
- Живая проверка (npm run dev):
  1. Проверить фоновый цвет блока реквизитов (видно на белом и чёрном)
  2. Кликнуть на иконку сворачивания — блок сворачивается
  3. Проверить, что в свернутом состоянии видна кнопка "Развернуть"
  4. Кликнуть "Развернуть" — блок раскрывается
  5. Проверить темный режим (фон видимо выделен)
