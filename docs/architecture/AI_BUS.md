# AI Bus

**Класть:** `docs/architecture/AI_BUS.md`

AI Bus --- протокол взаимодействия Архитектора (ChatGPT) и Claude.

## Роли

-   Архитектор --- принимает решения, проектирует архитектуру, пишет
    спринты.
-   Claude --- реализует код строго по ARP.

## Цикл

1.  Архитектор пишет Sprint.
2.  Claude реализует Step.
3.  Claude возвращает ARP.
4.  Архитектор проверяет.
5.  Только после OK следующий Step.

## Правила

-   Не менять scope.
-   Не делать скрытых улучшений.
-   Каждый Step независим.
-   build/lint обязательны.
