# Literary Studio — MVP Scope

## 1. Purpose

This document defines the boundaries of the first usable release of Literary Studio: what a
writer can do with it, and what is deliberately left out for now.

## 2. Included in MVP

### Book Management

- Create a book
- Edit book metadata
- Chapters
- Scenes

### Writing

- Scene editing
- Iterative writing
- Version-safe editing — a writer's previous work is never lost while editing

### Assistants

- Co-author
- Editor
- Critic
- Reader

### Review

- Manuscript review
- Iterative improvement

### Project Management

- Project persistence
- Multiple books

## 3. Explicitly Out of Scope

- Illustrations
- Merchandise
- Publishing integrations
- Marketing
- Payments
- Collaboration between multiple human users
- Mobile applications
- Voice interaction

## 4. MVP Success Criteria

- A writer can create a book.
- A writer can complete an entire manuscript.
- A writer can revise chapters.
- A writer can receive assistant feedback.
- A manuscript can reach Publishing Ready.

## 5. Design Principles

- Small but complete.
- Every feature supports writing.
- Avoid feature creep.
- Writer first.
- Quality over quantity.
