# Sprint-38-Step-01: Статистика слов и знаков

## Objective
Добавить статистику слов и знаков в нижнюю строку страницы. Пользователь видит в реальном времени:
- Количество слов в основном тексте книги
- Количество знаков с пробелами
- Количество знаков без пробелов

## Why Now
- Писатели отслеживают прогресс по объему
- Важная метрика для публикации (договоры часто указывают минимальное количество слов)
- Видимая фича для первого впечатления от Sprint-38

## Acceptance Criteria
- [ ] Статистика появляется в footer (нижняя строка)
- [ ] Показывает: "Слов: 5,234 | Знаков: 28,456 | Без пробелов: 23,102"
- [ ] Обновляется в реальном времени при редактировании
- [ ] Подсчитывает ТОЛЬКО основной текст (Scene.text), не метаданные
- [ ] Работает на всех устройствах (mobile/tablet/desktop)
- [ ] Dark mode поддерживается
- [ ] E2E тесты: проверить счетчики при редактировании

## Implementation

### Backend
- Функция подсчета слов/знаков в `workspaceController.ts`
- Сумма всех `scene.text` в активной книге
- Логика:
  - Слова: `text.trim().split(/\s+/).length` (если текст не пуст)
  - Знаки с пробелами: `text.length`
  - Знаки без: `text.replace(/\s/g, '').length`

### UI
- Footer компонент в `page.tsx` (под UnifiedBookView)
- Показывает статистику активной книги
- Format: "Слов: 1,234 | Знаков: 5,678 | Без пробелов: 4,890"
- Обновляется при изменении `focusedSceneKey` или `workspace.books[activeBook]`
