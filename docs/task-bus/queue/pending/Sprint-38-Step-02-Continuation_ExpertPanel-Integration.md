## Что нужно сделать

### Part A: ExpertPanel компонент (новый или улучшение AssistantPanel)

**Цель:** Показать личных экспертов как выбираемые опции рядом с системными помощниками (Соавтор, Редактор, Критик, Читатель).

**UI макет:**
```
┌─ Системные помощники ──────────────────────┐
│  📝 Соавтор  ✏️ Редактор  👁️ Критик  📖 Читатель
├─ Мои эксперты ────────────────────────────┤
│  💕 Романтический советник  [⚙️ 🗑️]
│  🎯 Эксперт для боевиков     [⚙️ 🗑️]
│  🌙 Фантазия помощник        [⚙️ 🗑️]
└────────────────────────────────────────────┘
```

### Part B: Функционал для каждого эксперта

- **⚙️ Редактировать** — открыть CustomExpertsDialog на вкладке "Мои" с выбранным экспертом для редактирования
- **🗑️ Удалить** — soft delete через DELETE /api/experts/:id, удалить из списка

### Part C: Интеграция с чатом

- При выборе эксперта (личного или системного) → подставить его `systemPrompt` в запрос к AI
- Сохранить выбранный эксперт в state (для текущей сессии)

### Part D: Остальные методы на raw SQL

Конвертировать оставшиеся методы repository на raw SQL (если Step-02 оставил):
- ⏳ updateExpert → PUT /api/experts/:id
- ⏳ loadPublicExperts → GET /api/experts/public
- ⏳ addPublicExpertToMe → POST /api/experts/public/:id
- ⏳ removePublicExpertFromMe → DELETE /api/experts/public/:id
