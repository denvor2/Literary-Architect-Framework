## Scope

Allowed paths:
- docs/vision/BOOK_LEVEL_ASSISTANTS_VISION.md (добавить один раздел

Forbidden paths:

## Objective

Добавить в конец docs/vision/BOOK_LEVEL_ASSISTANTS_VISION.md новый
раздел (после существующего последнего раздела):

```markdown
## 8. Серия книг (будущее, не срочно)

Product Owner: книги могут объединяться в серию — как правило, через
общих персонажей, сцены или общий контекст/мир. Пока не спроектировано
и не приоритизировано ("пока не срочно, но надо заложить и
зафиксировать").

**Статус:** идея зафиксирована, ничего не решено. Потенциальное
архитектурное следствие — Character и, возможно, другие сущности
могут в будущем существовать на уровне выше отдельной Book
(общие для нескольких книг), а не быть жёстко привязаны к одной
книге, как сейчас (Character сегодня — часть одного Workspace/Book).
Это не декларация решения — просто явно отмеченный кандидат на
пересмотр домен-модели, когда/если Product Owner решит взяться за
серии книг.
```

## Rules

- Только добавление в конец, ничего существующего не менять.

## Validation

- grep -c "Серия книг" docs/vision/BOOK_LEVEL_ASSISTANTS_VISION.md → 1
