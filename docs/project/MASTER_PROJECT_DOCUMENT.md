# Purpose

Этот документ является главным документом проекта Literary Studio.

Он предназначен для передачи проекта между сессиями ChatGPT,
разработчиками и архитекторами, фиксации текущего состояния проекта и
навигации по документации.

ИИ является помощником автора, но не заменяет автора.
