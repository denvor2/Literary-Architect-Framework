## Контекст

Сейчас Header.tsx содержит структуру меню (File/Edit/View/Help/About), но все пункты 
отключены и показывают "Скоро". 

Sprint 33 должна реализовать полное меню с базовыми действиями:

- **Файл:** Новая книга, Новая серия, Экспорт, Выход
- **Правка:** Отмена, Повтор, Вырезать, Копировать, Вставить
- **Вид:** Режим фокуса, Светлая/Тёмная тема
- **Помощь:** Горячие клавиши, О программе  
- **О программе:** Версия, авторские права, ссылки

## Scope

### Allowed paths (ТОЛЬКО):

- apps/studio/src/components/Header.tsx (реализовать меню + логика)
- apps/studio/src/app/page.tsx (передать callbacks для File/Edit actions)
- Новые компоненты dialogs (если нужны): `KeyboardShortcutsDialog.tsx`, `AboutDialog.tsx`, etc (в components/)

### Forbidden paths:

- apps/studio/src/domain/** (нет изменений доменной модели)
- apps/studio/prisma/** (нет изменений схемы)

## Objective

Реализовать полное меню приложения с действиями:

### 1. Файл (File)
- **Новая книга** → вызов onNewBook (передано в Header из page.tsx)
- **Новая серия** → вызов onCreateSeries
- **Экспорт** → пока disabled, "Скоро" (Phase 2)
- **Выход** → logout (для авторизованных) или нет-опа

### 2. Правка (Edit)
- **Отмена** → вызов undo() в Workspace (если undo есть, иначе disabled)
- **Повтор** → вызов redo() (если redo есть, иначе disabled)
- **Вырезать/Копировать/Вставить** → focus на editor, стандартные браузерные операции
  (или disabled если нет фокуса в textarea)
- **Удалить** → delete activeBook (confirm dialog)
- **Выделить всё** → Ctrl+A (браузерное)

### 3. Вид (View)
- **Режим фокуса** (checkbox) → toggle focusMode state в page.tsx, показать ✓ если активен
- **Светлая тема** (radio) → toggle classList или data-theme, показать ● если активна
- **Тёмная тема** (radio) → toggle classList, показать ● если активна
- **Развернуть на весь экран** → requestFullscreen() (disabled если не поддерживается)

### 4. Помощь (Help)
- **Горячие клавиши** → открыть KeyboardShortcutsDialog с полным списком
- **О программе** → открыть AboutDialog (версия, сайт, контакты)

### 5. О программе (About)
- Версия: "Literary Studio v1.0"
- Авторские права: "© 2026 [Author]"
- Ссылки: "Веб-сайт", "Документация", "GitHub" (если есть)

## Rules

1. **Undo/Redo:** если не реализованы в Workspace — disabled
2. **Cut/Copy/Paste:** работают через браузерные события, disabled если нет фокуса в editor
3. **Новая книга/серия:** используют существующие onNewBook/onCreateSeries callbacks
4. **Тема:** state в page.tsx (prefersDarkMode или themeMode), persist в localStorage
5. **Горячие клавиши:** список включает Ctrl+K (search), Ctrl+Z (undo), etc.
6. **Меню закрывается** после выбора пункта (кроме чекбоксов и radio)

## Validation

Живая проверка в браузере:

1. **Файл → Новая книга:**
   - Открывается диалог/создаётся новая книга (как сейчас при нажатии +)

2. **Файл → Новая серия:**
   - Открывается диалог создания серии

3. **Файл → Выход:**
   - Logout (если авторизован)

4. **Правка → Отмена (если было изменение):**
   - Последнее изменение отменяется (или disabled если undo недоступна)

5. **Вид → Режим фокуса:**
   - Toggle mode, показать ✓ в меню

6. **Вид → Светлая/Тёмная тема:**
   - Переключение темы, persist в localStorage
   - Page refresh сохраняет выбранную тему

7. **Помощь → Горячие клавиши:**
   - Открывается модальное окно со списком горячих клавиш
   - Закрывается при Escape или клике вне

8. **О программе:**
   - Показывает версию, авторские права, ссылки
